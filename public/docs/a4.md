##Assignment 4: Reaction Diffusion (Simulation #1)
For my reaction diffusion, I focused on controlling randomness for visual art. To begin, the simulation chooses a number of starting blobs between 5 and 15. Each blob then is assigned a random radius between 20 and 30 pixels. These blobs then begin to diffuse with the default values for feed, kill, and dt described in Karl Sims' tutorial, but users can vary the values with the TweakPane panel.
The dA and dB values vary randomly between clamped values throughout the canvas. The user is given control of this randomness with the panel: dA is a random value between Min_dA and 1, and dB is a random value between 0.15 and Max_dB. This random diffusion map is then fed to the compute shader, and each pixel uses its random diffusion rates. The diffusion map follows the criteria of a style map, though it is random rather than designed with a specific vision.
