## Assignment 1: Shader Live Coding

While reading the course reading, The Book of Shaders, I was fascinated by the Shapes section. Specifically, I liked the Piet Mondrian grid style and decided to construct my animation based on it. My animation uses time to animate the grid lines in loops using sine, cosine, and tangent functions. This looping was intentional, as I wanted some constant movements throughout the animation. While playing around with the trigonometric functions, I found that using tan for a line location would force it to speed off of the screen and then return back from the other side, which I loved. In my animation, time progresses the thickness of the grid lines and the strength of the background noise. This allows the audience to sense the effect of time, but still enjoy the constant looping of the lines.

I was technically interested in exploring how basic geometry can form an artistic piece, almost resembling a painting, and how using trigonometric functions can vary the animation. To accomplish this, I created helper methods and normalized the coordinate system and the time.
